# 投资者控制交易风险，提高交易稳定性的主要方法包括分散投资、运用止损策略、控制交易规模和避免情绪化交易等。 计算机交易和AI的出现，使我们实现上述方法和策略具有了可操作性：

## 1. 强大的智能扫描系统可以结合基本面的新闻和股票技术图表走势，从成千上万只股票中挑选出当日有潜在大幅波动可能的股票，根据不同板块选择出1-50只股票作为日内交易对象，并制定对冲策略，以降低潜在风险。

## 2. 机器人交易程序在日内可以同时执行买卖和管理10， 20 甚至50只以上的股票，无人工干涉，这种分散投资极大降低了日内交易风险。

## 3. 每笔自动交易预设了止盈点和止损点，这在瞬息万变的日内行情中非常重要。一旦点位触发，计算机立即准确无误地完成止盈和止损。

## 4. 机器人程序自动交易完美地避免情绪化交易，克服了交易的最大敌人-人性的贪婪和恐惧。

## 5. 机器人程序可以精确控制交易规模：当日内市场强势上涨时， 预设的大多数买单会被自动激活，交易规模自动放大；而当市场平稳或下跌时，大多数预设买单无法被激活，交易规模自动减小，避免潜在风险。

## 6. 因为不涉及长期持有股票，我们的盈利率的计算非常直观，计算方法如下：
       日盈利率%=（收盘后账户余额-开盘前账户余额）/开盘前账户余额   （假设使用所有资金进行日交易）
